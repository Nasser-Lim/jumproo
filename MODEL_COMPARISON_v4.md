# 🏆 모델 성능 비교 리포트 (v1.0 ~ v6.0)

주가 급등 예측기의 버전별 성능과 특성을 비교 정리한 문서입니다.

## 📊 성능 종합 비교

| 버전 | 모델 아키텍처 | 핵심 특징 | Precision (15% 급등) | 평균 수익률 (5일) | 리스크 | 신호 빈도 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **v1.0** | Chronos (T5) | 단변량 (가격) | 11.9% | +5.42% | 높음 | 높음 |
| **v2.0** | PatchTST | 다변량 (가격+거래량) | 13.9% | +6.26% | 보통 | 높음 |
| **v3.0** | PatchTST + MC Dropout | 불확실성 추정 | 12.1% | +4.99% | **최저 (7.2%)** | 낮음 |
| **v4.0** | 앙상블 (5-모델) | 체급별 + 기술적 + 지수 | 20.5% | +8.62% | 보통 (8.9%) | 선별적 |
| **v6.0 Baseline** | EVT + Hawkes | 극단값 이론 + 호크스 과정 | 21.7% | +11.44% | 낮음 | 매우 선별적 |
| **v6.0 Full** | **HMM + EVT + TDA + PathSig + Hawkes** | **5단계 통계 파이프라인** | **24.0%** | **+14.47%** | 낮음 | **매우 선별적** |

> v6.0 Precision/수익률은 threshold 0.60 (STRONG_BUY) 기준. v1~v4는 각 모델의 최적 threshold 기준.

## 🧐 버전별 상세 분석

### v1.0: 베이스라인 (Chronos)
- **컨셉**: 주가 차트를 언어 토큰처럼 학습
- **장점**: 사전학습(10M+ 시계열)으로 일반적인 파동 패턴 포착 우수
- **단점**: 거래량, 시장 맥락을 볼 수 없음. "가짜 돌파"를 구분하지 못함

### v2.0: 눈뜨기 (PatchTST + 거래량)
- **컨셉**: "거래량이 가격에 선행한다". 로그 거래량 채널 추가
- **결과**: Precision +2%p 향상. *거래량 없는 가격 상승*이 함정임을 학습

### v3.0: 방패 (리스크 관리)
- **컨셉**: "돈을 잃지 말자". Monte Carlo Dropout으로 불확실성 측정
- **결과**: 신호를 v2.0의 1/7로 대폭 축소
- **활용**: **하락장 보호**. 시장이 불안할 때 극도로 확신이 있을 때만 신호 발생

### v4.0: 저격수 (앙상블 + 매크로)
- **컨셉**: "집단 지성"
  - **앙상블**: 5개 모델 투표, 4개 이상 동의(Vote >= 0.8)하면 신호
  - **매크로**: KOSPI/KOSDAQ 지수 수준 인식
  - **마이크로**: RSI/MACD로 최적 진입점 탐색
- **결과**: 당시 최고 Precision (20.5%)
  - 소형주: 21.1% Precision ("원석 발굴기")
  - 대형주: 22.2% Precision (Vote 1.0 기준, "추세 탑승기")
- **활용**: **적극 트레이딩**. v4.0이 종목을 지목하면 **5개 중 1개가 1주일 내 15% 이상 급등**

### v6.0 Baseline: 순수 통계 (EVT + Hawkes)
- **컨셉**: "GPU 없이 수학으로 승부". 딥러닝 완전 제거, 순수 통계 파이프라인
  - **극단값 이론(EVT/GPD)**: 수익률 꼬리 분포에서 급등 확률 추정 (가중치 40%)
  - **호크스 과정**: 급등 이벤트의 군집 패턴 학습 (가중치 60%)
  - **국면 게이트**: 변동성+거래량 기반 이진 게이트 (급등 준비 안되면 x0.1 페널티)
- **결과**: 80,673건 백테스트
  - Threshold 0.40: 1,723 신호, Precision 18.57%, 평균 수익률 +7.96%
  - Threshold 0.60: 115 신호, Precision 21.74%, 평균 수익률 +11.44%
- **강점**: GPU 불필요, 실시간 계산 가능, 해석 가능한 지표

### v6.0 Full Pipeline: HMM + TDA + 경로서명 통합
- **컨셉**: 3개 추가 모듈로 파이프라인 확장
  - **HMM 국면 판별**: 은닉 마르코프 모델로 시장 국면을 확률적으로 분류, 이진 게이트를 **Soft Gate**로 교체
  - **TDA 위상 분석**: 가격 시계열의 위상 구조 분석 (참고 지표)
  - **경로 서명**: 다변량 경로의 반복 적분으로 동역학 포착 (참고 지표)
- **결과**: 동일 80,673건 백테스트

| Threshold | Baseline 신호 | Full 신호 | Baseline Prec | Full Prec | Delta Prec | Baseline 수익률 | Full 수익률 | Delta 수익률 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 0.40 | 1,723 | 653 | 18.57% | 18.22% | -0.35%p | +7.96% | +7.62% | -0.34%p |
| 0.50 | 695 | 265 | 20.58% | **21.51%** | **+0.93%p** | +9.36% | +8.55% | -0.80%p |
| 0.60 | 115 | 50 | 21.74% | **24.00%** | **+2.26%p** | +11.44% | **+14.47%** | **+3.03%p** |

- **핵심 발견**:
  - HMM Soft Gate가 고확신 구간(0.50+)에서 true positive를 추가 포착하여 precision 향상
  - TDA/경로서명은 급등 vs 비급등 간 판별력이 거의 없음 (delta ~0) → 점수에는 미반영, 분석용으로만 포함
  - **STRONG_BUY 신호: Precision 24.0%, 평균 수익률 +14.47%** (역대 최고)

## 🎯 최종 운용 전략

1. **주력 전략 (v6.0 Full)**: STRONG_BUY 신호(threshold >= 0.60) 발생 시 진입. **4개 중 1개가 5일 내 15% 이상 급등**, 평균 수익률 +14.47%.
2. **보조 전략 (v4.0)**: v6.0이 NEUTRAL일 때 v4.0 앙상블(Vote >= 0.8)로 교차 검증.
3. **방어 전략 (v3.0)**: 하락장(KOSPI < 52주 저점 + 10%)에서는 v3.0 불확실성 점수로 추가 필터링.
4. **체급별 배분**:
   - 소형주 40%: 높은 잠재력, v6.0 STRONG_BUY 집중
   - 대형/중형주 60%: 안정성, v6.0 BUY + v4.0 교차검증
